# AI Agents Documentation

This document describes the responsibilities, inputs, outputs, and AI techniques used by each agent within the AI Scientific Content Generation System.

---

# Agent 1 — Clinical Research Agent

## Purpose

Analyze scientific literature and extract reliable, evidence-based information related to the requested healthcare topic.

## Responsibilities

- Analyze scientific publications
- Identify clinically relevant findings
- Prioritize high-quality evidence
- Summarize biological mechanisms
- Detect conflicting evidence

## Inputs

- Scientific papers
- Clinical guidelines
- User topic
- Research objective

## Outputs

- Structured evidence summary
- Main findings
- Biological mechanisms
- Evidence level
- Clinical references
- Research package for downstream agents

## AI Techniques

- Prompt Engineering
- Context Engineering
- Structured Outputs
- Rule-Based Reasoning

---

# Agent 2 — Clinical Knowledge Translation Agent

## Purpose

Transform scientific evidence into educational language that is understandable for non-specialists while preserving scientific accuracy.

## Responsibilities

- Simplify complex concepts
- Generate analogies
- Generate metaphors
- Produce educational explanations
- Extract practical insights

## Inputs

- Structured evidence package
- Clinical findings
- Target audience profile

## Outputs

- Educational explanations
- Analogies
- Metaphors
- Practical insights
- Simplified scientific concepts

## AI Techniques

- Prompt Chaining
- Context Conditioning
- Knowledge Translation
- Structured Outputs
